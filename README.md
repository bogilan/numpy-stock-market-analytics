# Stock Market Analytics with NumPy

A portfolio project that simulates stock market data and performs financial analysis using NumPy and Matplotlib.

## Project Overview

This project generates synthetic stock price data for multiple companies using a random walk model and applies common financial analytics techniques.

The goal is to demonstrate practical NumPy skills used in data analytics, quantitative finance, and data science workflows.

---

## Technologies Used

- Python
- NumPy
- Matplotlib

---

## Features

### Stock Price Simulation

- Simulates 365 trading days
- Multiple stocks tracked simultaneously
- Random walk model based on daily returns
- Reproducible results using NumPy random seed

### Financial Analysis

- Daily returns calculation
- Total returns
- Volatility analysis
- Moving averages
- Performance ranking
- Correlation analysis

### Data Visualization

- Stock price trends
- Moving average overlays
- Volatility comparison charts
- Correlation heatmaps

---

## NumPy Concepts Demonstrated
